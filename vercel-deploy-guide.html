🚀 Guía Completa: Deployar en Vercel

Para estudiantes de Front-End

¿Qué es Vercel?

Vercel es una plataforma de hosting especializada en aplicaciones web modernas. Es gratuita para proyectos personales y estudiantes, y permite deployar tu código directamente desde GitHub en segundos. Es la opción ideal para compartir tus proyectos de clase.

✨ Ventajas de Vercel

  • Gratuito para estudiantes: Sin costo para proyectos públicos
  • Integración con GitHub: Deployar es automático al hacer push
  • Rápido y confiable: Tu sitio carga en millisegundos
  • SSL automático: Tu sitio es seguro (HTTPS)
  • Dominios personalizados: Puedes usar tu propio dominio
  • Fácil de usar: No necesitas línea de comando (aunque puedas usarla)

📋 Guía Paso a Paso

Paso 1: Crear Cuenta en Vercel

1 Abre vercel.com

Haz clic en "Sign Up" (Registrarse)

2 Elige una opción de registro. Recomendado: usar tu cuenta de GitHub

Esto facilita mucho todo el proceso

3 Autoriza Vercel para acceder a tu GitHub

Vercel necesita permiso para leer tus repositorios

Paso 2: Importar tu Proyecto desde GitHub

1 En el dashboard de Vercel, haz clic en "Add New" → "Project"

2 Verá una lista de tus repositorios de GitHub

Busca y selecciona el repositorio que quieres deployar (ej: itfs-front-end)

3 Configura el proyecto

  • Project Name: Dale un nombre (por defecto es el del repo)
  • Framework: Selecciona Next.js si uses React, o Other si es HTML/CSS/JS
  • Root Directory: Déjalo en blanco (.) si el proyecto está en la raíz

4 Haz clic en "Deploy"

Vercel empezará a construir y desplegar tu proyecto

Paso 3: Espera a que se Complete

Vercel mostrará una barra de progreso. En 1-2 minutos:

  • ✅ Tu proyecto estará en vivo
  • ✅ Tendrás una URL pública (ej: tu-proyecto.vercel.app)
Vercel crea automáticamente una URL para tu proyecto. Cada vez que hagas un push a GitHub, Vercel redesplegará automáticamente tu sitio con los cambios más recientes.

🎯 Después del Deploy

Tu Nuevo URL (Dominio Vercel)

Vercel te asigna automáticamente una URL como:

tu-proyecto.vercel.app

Comparte este URL con tu profesor/a y compañeros. ¡Tu proyecto está vivo en internet! 🌐

¿Qué Pasa Cuando Hago Push a GitHub?

Vercel está configurado por defecto para redeploy automático:

  1. Modificas archivos en tu computadora
  2. Haces git add, git commit, git push
  3. Vercel detecta el cambio automáticamente
  4. Tu sitio se actualiza en segundos
No necesitas hacer nada más después de hacer push. Vercel se encarga de todo automáticamente.

Ver el Progreso del Deploy

  1. Ve a tu dashboard de Vercel
  2. Haz clic en tu proyecto
  3. Irás a una página que muestra todos tus deployments
  4. Puedes ver el historial y el estado de cada deploy

⚙️ Configuraciones Útiles (Opcional)

Cambiar el Nombre de tu Dominio

  1. Ve a tu proyecto en Vercel
  2. Haz clic en "Settings" en la barra superior
  3. Ve a "Domains"
  4. Haz clic en "Edit" al lado de tu dominio
  5. Cambia el nombre (ej: de m9-proyecto.vercel.app a mi-tienda-online.vercel.app)

Agregar un Dominio Personalizado

Si tienes tu propio dominio (comprado en GoDaddy, Namecheap, etc.):

  1. En "Domains", haz clic en "Add"
  2. Escribe tu dominio personalizado
  3. Sigue las instrucciones para apuntar el DNS
Configurar dominios personalizados es más avanzado. Vercel tiene buena documentación si lo necesitas.

Proteger tu Sitio con Contraseña

Si tu proyecto es privado y solo quieres que lo vea tu profesor:

  1. Ve a "Settings"
  2. Busca "Password Protection"
  3. Actívalo e ingresa una contraseña

🔧 Solución de Problemas

❌ Mi sitio muestra un error (Error 404 o blank page)
Posibles soluciones:
  • Verifica que tu archivo principal sea index.html
  • Si usas una carpeta específica, indica el "Root Directory" en Vercel
  • Haz clic en "Redeploy" en Vercel para forzar una nueva construcción
❌ Los cambios de GitHub no aparecen en Vercel
Posibles soluciones:
  • Espera 1-2 minutos después del push (Vercel tarda en actualizar)
  • Recarga la página con Ctrl+F5 (limpia el caché)
  • En el dashboard de Vercel, haz clic en "Redeploy" manualmente
❌ Mi sitio luce diferente en Vercel que en mi computadora
Posibles causas y soluciones:
  • Las rutas de archivos en tu HTML podrían ser incorrectas
  • Usa rutas relativas en lugar de absolutas (ej: ./styles.css no C:/proyectos/styles.css)
  • Verifica que todos los archivos (CSS, JS, imágenes) estén en el repositorio
❓ ¿Cuánto tarda el deploy?
Los primeros deployments suelen tardar 2-3 minutos. Los siguientes son más rápidos (30 segundos a 1 minuto) porque Vercel cachea los resultados.

📝 Ejemplo Real: Tu Proyecto de Clase

Supongamos que tienes un proyecto de React que quieres compartir:

Paso 1: Push a GitHub

git add . git commit -m "Proyecto M12 lista" git push origin main

Paso 2: En Vercel

  1. Dashboard → "Add New" → "Project"
  2. Selecciona tu repo
  3. Framework: "Next.js" (o lo que uses)
  4. Haz clic en "Deploy"

Paso 3: Listo

En 2 minutos tu proyecto estará en: https://tu-proyecto.vercel.app 🎉

Cada vez que hagas push a GitHub, tu sitio en Vercel se actualizará automáticamente sin que hagas nada.
