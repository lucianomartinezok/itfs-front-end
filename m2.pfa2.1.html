Propuesta B: Elige una página web que utilices habitualmente y utiliza la herramienta "inspeccionar" del navegador para observar el código HTML. Anota las etiquetas HTML que vayas identificando y determina cuáles te resultan conocidas y cuáles no.

Análisis de Código HTML (Propuesta B)

Ejemplo de Estructura

Etiquetas conocidas:

  • <header> - Encabezado
  • <nav> - Navegación
  • <main> - Contenido principal
  • <footer> - Pie de página

Etiquetas menos familiares:

  • <article> - Representa un contenido independiente
  • <aside> - Información relacionada (barra lateral)
  • <section> - Agrupación temática

Análisis de etiquetas realizado con herramientas de inspección.
