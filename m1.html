Resumen - Mosaico 1

Exploramos los fundamentos del desarrollo frontend, desde la evolución de Internet hasta las herramientas para crear experiencias web.

1. Fundamentos de Internet y la Web

  • Conceptos clave: Diferencia y evolución de Internet y la Web.
  • Infraestructura: TCP/IP, DNS, dominios y URLs.
  • Modelo Cliente-Servidor: Base para la comunicación en la red.

2. Lenguajes y Procesos Web

  • Herramientas esenciales: HTML, CSS, JavaScript y la historia de HTML.
  • Renderizado: Cómo los navegadores transforman datos en interfaces.
  • Aplicaciones web: Ventajas y relevancia actual.

3. Desarrollo Frontend y Herramientas

  • Rol del Frontend: Importancia del lado del cliente frente al backend.
  • Herramientas prácticas: Uso de Visual Studio Code y herramientas del navegador.
  • Páginas: Diferencias entre sitios estáticos y dinámicos.
Recuerda que este contenido puede ser útil para repasar más adelante algunos conceptos. Guarda este resumen como referencia para cuando necesites refrescar tus conocimientos.
