🚀 Resumen del Módulo

El M5 se compone de conceptos de JavaScript moderno: ECMAScript 7, Arrays y sus métodos avanzados (bucles, comprobaciones, transformadores, filtros, búsquedas, acumuladores, desestructuración y reestructuración).

También incluye las APIs de JavaScript (nativas y externas, formularios, validación, Media API) y la asincronía (callbacks y promesas).

2 ECMAScript

Estándar que define el lenguaje JavaScript. Cada versión suma sintaxis y nuevas features.

2.1 Versiones

  • ES5 (2009): base histórica.
  • ES6 / 2015: let/const, arrow fns, clases, módulos, template literals.
  • ES2016+: **, includes(), async/await, ?. (optional chaining).

2.2 Crossbrowser

  • Verificar soporte en caniuse.com.
  • Transpilar con Babel.
  • Usar polyfills para funciones faltantes.

3 Arrays

Métodos de alto nivel que reemplazan al for clásico y hacen el código más declarativo.

3.1 · 3.2 Funciones y bucles

  • forEach() — recorre sin retornar.
  • for...of — bucle moderno sobre iterables.

3.3 Comprobaciones

  • every() — ¿todos cumplen?
  • some() — ¿alguno cumple?

3.4 Transformadores y filtros

  • map() — transforma cada elemento.
  • filter() — devuelve los que cumplen.

3.5 Búsquedas

  • find(), findIndex(), includes(), indexOf().

3.6 Acumuladores

  • reduce() / reduceRight() — reducen a un único valor.

3.7 · 3.8 Desestructurar e intercambiar

  • const [a, b] = [1, 2]
  • Swap: [x, y] = [y, x]

3.9 · 3.10 Spread / Rest / Group

  • Spread: [...arr1, ...arr2]
  • Rest: function f(...args) { }
  • Agrupar: Object.groupBy() (ES2024).

3.11 Reestructuración

  • flat() — aplana arrays anidados.
  • flatMap() — map + flat en un paso.

4 Set / Map

Set guarda valores únicos. Map guarda pares clave-valor (con claves de cualquier tipo).

  • 4.1 size — cantidad de elementos.
  • 4.2 add(v) — añade un elemento.
  • 4.3 has(v) — ¿existe?
  • 4.4 delete(v) — borra un elemento.
  • 4.5 clear() — vacía el conjunto.
  • 4.6 Conversión: [...miSet] · new Set(arr) (elimina duplicados).
